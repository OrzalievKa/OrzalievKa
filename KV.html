Что такое «система контроля версий»? - это инструмент для отслеживания изменений в файлах и координации работы над проектом.

К какому типу VSC относится? В чем главное отличие от других типов? - Главное отличие от централизованных VCS в том, что каждый разработчик хранит полную копию репозитория, а не только последние изменения. В чем заключается преимущество использование Git? - Работа без подключения к сети, высокая скорость, эффективное слияние веток, надёжность хранения данных Какие команды Git вы знаете? - git init – инициализация репозитория, git add – добавление изменений в индекс, git commit – фиксация изменений, git push – отправка изменений в удалённый репозиторий В чем отличие Git и Git Hub? - Git – инструмент для работы с версиями кода локально. GitHub – облачный сервис для хранения и совместной работы с Git-репозиториями.
